# 🎁Gift Suggestion System (Python)

## Description
This is a simple Python project that suggests gifts based on:
- Budget
- Occasion
- Person

##  Features
- Budget-based suggestions
- Occasion selection (Birthday, Anniversary, Eid, Festival)
- Easy to use

##  Technologies Used
- Python

##  How to Run
1. Open terminal
2. Run:
   python your_file_name.py

##  Output Example
Displays gift suggestions based on input.
